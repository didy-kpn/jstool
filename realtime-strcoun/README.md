# Realtime Strcount

HTML + CSS + JavaScript だけで動作する、ブラウザ向けのシンプルなリアルタイム文字数カウントツールです。

## 概要

入力テキストをリアルタイムで集計し、右側パネルに以下を表示します。

- 文字数
- 文字数（全ての空白なし）
- 行数
- 段落数
- 原稿用紙枚数（400字詰め換算、整数）

## 使い方

1. `index.html` をブラウザで開きます。
2. 左側の `テキスト入力` に文字を入力または貼り付けます。
3. 右側の `集計結果` が自動で更新されます。

補助操作:

- `クリア`: 入力内容を初期化し、集計値をリセット

## 使用技術

- HTML5
- CSS3
- Vanilla JavaScript

## 仕様

### 入力仕様

- 入力対象は1つのテキストエリア
- 入力ソース
  - 手入力
  - 貼り付けテキスト
- `input` イベント発生時に即時で再集計を実行

### 集計計算仕様

- 文字数: 改行を含む全文字数（`Array.from(text).length`）
- 文字数（全ての空白なし）: 空白文字（半角・全角・改行・タブ等）を除外した文字数
- 行数: 空文字は 0、非空文字列は `\n` 区切りの行数
- 段落数: 空行区切り（`/\n\s*\n+/`）で段落を判定
- 原稿用紙枚数: `空白なし文字数 / 400` を整数切り上げ（0文字は0）

### 表示仕様

- 2カラムレイアウト
  - 左: テキストエディタ
  - 右: 集計結果カード
- 数値は `ja-JP` ロケールで3桁区切り表示
- 原稿用紙枚数は小数点なしの整数表示

### レスポンシブ仕様

- 画面幅が狭い場合、2カラムから1カラムへ切り替え
- モバイル表示時は入力エリアの最小高さを縮小

## 注意事項

- 文字数計算はブラウザ上で実行され、サーバ送信は行いません。
- 非常に大きなテキストでは、入力時の描画更新が重くなる場合があります。
